# FuelInsight AI v2.0

Website nghiên cứu khoa học hiện đại cho đề tài dự báo giá dầu DO 0,05S-II và quản trị chi phí logistics.

## Chạy local

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy

Upload toàn bộ thư mục lên GitHub rồi deploy bằng Streamlit Community Cloud.

Main file path:

```text
app.py
```
